[English](README_EN.md) | [中文](README.md)

---

## ✨ 项目简介

**OpenVisio** 是一款自然语言驱动的 Visio 图表生成工具。用户只需输入一段自然语言描述，AI 即可自动理解语义、识别节点与关系，并生成专业、可编辑的 `.vsdx` 文件。

无需画图技能，无需手动拖拽，只需一次回车。

---

## 🚀 核心功能

| 功能 | 描述 |
|------|------|
| **流程图** | 将工作流程拆解为清晰的步骤与分支判断 |
| **时序图** | 展示多角色之间按时间顺序的交互与消息往来 |
| **功能组成图** | 将系统功能模块逐层拆解为树状结构 |
| **关系图** | 描述组件之间的接口提供与依赖关系 |
| **架构图** | 用分层结构呈现系统各层级与组成模块 |
| **泳道图** | 按角色分泳道，清晰呈现跨部门协作流程 |

## 📦 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (Vanilla)
- **设计风格**：深色科技风，渐变紫蓝配色
- **响应式**：支持桌面端与移动端自适应

## 🛠️ 本地开发

### 环境要求

- 现代浏览器（Chrome / Firefox / Edge / Safari）
- 本地服务器（可选，用于预览）

### 快速开始

```bash
# 克隆仓库
git clone <repository-url>
cd openvisio

# 直接在浏览器中打开
open index.html

# 或使用本地服务器
npx serve .
```
